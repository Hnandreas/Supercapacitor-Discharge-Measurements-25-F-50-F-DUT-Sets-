# Measurement Data – 25 F Supercapacitors

This folder contains measurement data, results and related files for supercapacitor discharge tests using 25 F units from six different manufacturers. 

## Contents

- [Dataset Overview](#dataset-overview)
- [Discharge Curve Gallery](#discharge-curve-gallery)

- [Important Notes](#important-notes)
- [Related Links](#related-links)

## Dataset Overview

This dataset (Multi-Manufacturer Set) includes six manufacturers, each represented by three 25 F devices from the same batch to ensure statistical consistency:

- Eaton
- Kyocera
- Maxwell
- Sech
- Vishay
- Würth Elektronik

Each device was tested at three discharge current levels (IEC 62391-1 Method 1A Classes 2, 3, and 4) representing progressively increasing current conditions. For detailed information about the experimental setup and measurement procedures, see the [Main Repository README](../README.md#experimental-setup-and-methodology).
